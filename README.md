### Devtools
* element-ui 1.2.8
* jQuery 3.2.1
* Font Awesome 4.7.0
* Vue.js 2.2.6

### 编码规范
* css变量命名：元素名_属性名
* js变量命名：_分隔
* js函数命名：驼峰法
* HTML中class、id名：-分隔
* 版本更新遵循x.y.z原则：只修改bug则迭代z、页面有较大变动则迭代y、页面有颠覆性的变动则迭代x
